# JNB LLC Website

**jnbllc.com** — Bookkeeping, Tax Preparation, Fractional CFO & Financial Systems

## Stack
- React 19 + TypeScript
- Tailwind CSS 4
- Vite
- Wouter (routing)
- Framer Motion (animations)
- Deployed on Netlify

## Local Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Assets
Place your actual logo and headshot in `/public/`:
- `logo.png` — JNB LLC owl logo
- `headshot.jpg` — Sara's headshot photo

## Deploy
Push to GitHub → Netlify auto-deploys from `main` branch.
Build command: `npm run build`
Publish directory: `dist`
